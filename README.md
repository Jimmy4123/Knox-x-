# AnimeSAO Pro

Una experiencia premium de streaming de anime con recomendaciones potenciadas por Inteligencia Artificial (Gemini), gestión inteligente de biblioteca y automatización de búsqueda.

## 🚀 Características

- **Asistente AI**: Recomendaciones personalizadas utilizando el modelo Gemini de Google.
- **PWA (Progressive Web App)**: Instalable en dispositivos móviles y escritorio para uso offline.
- **Biblioteca Inteligente**: Guarda tus animes favoritos y mantén el historial.
- **Interfaz Premium**: Diseño moderno, oscuro y optimizado para una navegación fluida.
- **Búsqueda Avanzada**: Encuentra animes por género, popularidad o lanzamientos recientes.

## 🛠️ Tecnologías

- **Frontend**: React 19, Tailwind CSS 4, Motion (framer-motion).
- **Backend**: Express.js (Node.js).
- **AI**: Google Gemini API.
- **Build Tool**: Vite.
- **Lenguaje**: TypeScript.

## 📦 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/animesao-pro.git
   cd animesao-pro
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Configurar variables de entorno:**
   Crea un archivo `.env` basado en `.env.example` y añade tu `GEMINI_API_KEY`.

4. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

5. **Construir para producción:**
   ```bash
   npm run build
   npm start
   ```

## 🔑 Configuración de AI
Para usar el asistente de recomendaciones, necesitas una API Key de Google Gemini. Puedes obtenerla en [Google AI Studio](https://aistudio.google.com/).
